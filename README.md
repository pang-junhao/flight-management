# Flight Management System (C#)

## Overview

This project is a console-based Flight Management System developed in C#. It was created as part of an Object-Oriented Programming (COS20007) assignment.

The system allows administrators to manage flights and passengers to search and book flights through a menu-driven interface.

## Features

### Admin Functions

* Add Flights
* View Flights
* Update Flights
* Remove Flights
* Search Flights

### Passenger Functions

* Search Flights
* Book Flights
* View Bookings
* Cancel Bookings

## Object-Oriented Programming Concepts
